# NF4 Log Analyzer

NetFUNNEL 4 로그 파일을 분석하는 웹 애플리케이션입니다.

NF4 액세스 로그를 업로드하거나 서버 로컬 경로를 지정하여, 유량제어 세션 통계(진입/대기/완료/이탈), 상태코드 분포, 처리시간 통계, IP별 사용자 추적 등을 시각적으로 확인할 수 있습니다. NF 서버가 2대 이상인 경우 멀티서버 통합 분석도 지원합니다.

## 주요 기능

- **로그 파일 업로드 또는 서버 로컬 경로 지정** (3GB+ 대용량 스트리밍 처리)
- **멀티서버 통합 분석**: 서버 2대 이상의 로그를 한 화면에서 동시 분석. 통합/서버별 탭 전환 지원
- **KPI 대시보드**: 진입 IP 수, 요청/대기/완료/이탈 사용자 수, 이탈률
- **처리시간 통계**: 키 발급 ~ 키 반납(5004) 구간의 평균/최소/최대. 타임아웃 초과 세션은 제외
- **RPS 간격 분석 및 Hold 정책 분석**
- **대기시간 통계**: 진입 성공 vs 이탈 사용자 비교
- **비정상 상태코드 상세 목록 및 CSV 다운로드**
- **IP별 사용자 추적**: 시간대 필터, 진입 유형/상태 뱃지, 키 발급/대기 종료/키 반납 시각 표시. 멀티서버 시 서버 라벨 표시
- **대시보드 (IP Top)**: 요청/대기/이탈 최다 IP, 시간대 필터, IP 클릭 시 사용자 추적 연동

## 실행 방법

### 요구사항

- Python 3.12+

### 1. 압축 해제

공유드라이브에서 다운로드한 zip 파일을 원하는 위치에 압축 해제합니다.

### 2. 가상환경 생성 및 활성화

```bash
cd log_analyze

python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
```

### 3. 패키지 설치

```bash
pip install -r requirements.txt
```

### 4. DB 초기화

```bash
python manage.py migrate
```

### 5. 서버 실행

```bash
python manage.py runserver
```

브라우저에서 http://127.0.0.1:8000 에 접속합니다.

## 사용 방법

### 단일 서버 분석

1. **서버 로그 파일** 항목에서 파일 경로를 입력하거나 파일을 업로드합니다.
2. 필요 시 프로젝트 ID, 세그먼트 ID, 시간 범위를 입력합니다.
3. **처리시간 최대 타임아웃(초)**를 설정합니다 (기본값: 20초). 이 값을 초과하는 세션은 처리시간 통계에서 제외됩니다.
4. **결과 출력** 버튼을 클릭합니다.

### 멀티서버 분석

1. **+ 서버 추가** 버튼으로 서버를 원하는 대수만큼 추가합니다.
2. 각 서버 항목에 라벨(예: 서버1, NF-AP-01)과 파일 경로 또는 파일을 입력합니다.
3. **결과 출력** 버튼을 클릭하면 전체 통합 통계와 서버별 탭이 함께 제공됩니다.

### 사용자 추적

1. 로그 분석 완료 후 **사용자 추적** 탭으로 이동합니다.
2. IP를 입력하고 검색합니다 (시간대 필터 선택 가능).
3. 해당 IP의 모든 세션 목록과 요약 카드(전체 세션 수, 총 대기시간, 최초/마지막 접속)를 확인합니다.
4. 각 세션의 진입 유형(즉시진입/대기후진입), 상태(완료/대기이탈/진입이탈/진행중), 타임스탬프를 색상 뱃지로 확인할 수 있습니다.

## API 엔드포인트

| 엔드포인트 | 메서드 | 설명 |
|-----------|--------|------|
| `/` | GET | 웹 UI |
| `/api/analyze/` | POST | 파일 업로드 분석 (단일 서버) |
| `/api/range/` | POST | 파일 업로드 시간범위 조회 |
| `/api/analyze_path/` | GET | 서버 로컬 파일 경로 분석 |
| `/api/range_path/` | GET | 서버 로컬 파일 시간범위 조회 |
| `/api/analyze_multi/` | POST | 멀티서버 통합 분석 (파일 업로드 + 경로 혼합 지원) |
| `/api/trace/` | GET | IP별 사용자 추적 (시간대 필터 지원) |
| `/api/dashboard_filter/` | GET | 대시보드 IP Top 시간대 필터 조회 |
| `/api/csv/` | POST | CSV 다운로드 |
